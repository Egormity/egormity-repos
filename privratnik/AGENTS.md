# Privratnik Project Group Navigation

This folder contains the Privratnik web cabinet and mobile app.

## Repositories

- `lk/` - Vite/React personal account web client with role-based routing, generated API clients, Redux, React Query, and dashboard-style pages.
- `privratnik-frontend/` - React Native mobile app for barriers, maps, cameras/video streams, passes, chat, payments, settings, and realtime access/control flows.

## Agent Rule

For implementation work, enter the relevant repository and read its local `AGENTS.md`. Repository `README.md` files are only short developer quickstarts.
