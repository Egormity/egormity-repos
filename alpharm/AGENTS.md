# Alpharm Project Group Navigation

This folder contains three related Alpharm repositories.

## Repositories

- `alpharm-web/` - main Alpharm web CRM/front office application built with Vite, React, TanStack Router, TanStack Query, Zustand, and generated Swagger-style API modules.
- `alpharm-admin/` - admin/back office web application for managing company structures, employees, libraries, applicants, analytics, and operational dictionaries.
- `alpharm-mobile/` - React Native mobile application for field work: visits, schedules, contacts, organizations, offline Realm storage, and sync flows.

## Agent Rule

For implementation work, enter the relevant repository and read its local `AGENTS.md`. Repository `README.md` files are only short developer quickstarts.
