# On Fashion Project Group Navigation

This folder contains the On Fashion website and native mobile wrappers.

## Repositories

- `online-fashion.ru-bitrix/` - main Bitrix website/template repository for `online-fashion.ru`; app-facing template work lives in `local/templates/app`.
- `on-fashion-android/` - native Android WebView shell for `https://www.online-fashion.ru`.
- `on-fashion-ios/` - native iOS SwiftUI/WKWebView shell for `https://www.online-fashion.ru?utm_source=app`.

## Agent Rule

For implementation work, enter the relevant repository and read its local `AGENTS.md`. Repository `README.md` files are only short developer quickstarts.
