# Music of Health Project Group Navigation

This folder contains native mobile wrappers for the Music of Health web product.

## Repositories

- `music-android/` - native Android WebView shell that loads `https://healthmusic.ru` and includes a foreground service.
- `music-ios/` - native iOS SwiftUI/WKWebView shell that loads `https://healthmusic.ru`.

## Agent Rule

For implementation work, enter the relevant platform repository and read its local `AGENTS.md`. Repository `README.md` files are only short developer quickstarts.
