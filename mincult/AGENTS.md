# Mincult Project Group Navigation

This folder contains the Culture Ural public web and mobile applications.

## Repositories

- `mincult-frontend/` - Next.js public website for Culture Ural content: events, places, news, articles, special projects, about page, and event submission.
- `mincult-mobile/` - Expo/React Native mobile app for the same content ecosystem: maps, posters, news, chatbot/profile flows, push notifications, app update checks, and location-aware features.

## Agent Rule

For implementation work, enter the relevant repository and read its local `AGENTS.md`. Repository `README.md` files are only short developer quickstarts.
