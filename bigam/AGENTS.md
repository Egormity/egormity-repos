# bigam Agent Navigation

This folder contains one or more working repositories for the bigam area.
Before making changes, open the target repository and follow its local AGENTS.md and docs.

## Repositories

- `bigam-webview/` — https://gitlab.com/ruyou/bigam-webview.git
- `bigam_v5/` — https://gitlab.com/ruyou/bigam_v5.git

## Active Development Scope

- Work **only** in `bigam-webview/` unless the user explicitly requests otherwise.
- `bigam_v5/` is the old React Native application. Treat it as read-only reference material when the behavior or business logic of the webview requires clarification.
- Do not implement fixes or features in `bigam_v5/` based solely on a request concerning Bigam or the webview.
