# Podvijnik Project Group Navigation

This folder contains the Podvizhnik radio mobile app and backend API.

## Repositories

- `podvijnik-frontend/` - React Native/Expo mobile app for radio playback, broadcasts, schedules, downloads, favorites, live questions, and notifications.
- `podvizhnik-backend-php/` - Yii2 Advanced PHP backend that serves API endpoints, rubrics, schedule, feedback, docs, push notifications, and admin/backend surfaces.

## Agent Rule

For implementation work, enter the relevant repository and read its local `AGENTS.md`. Repository `README.md` files are only short developer quickstarts.
