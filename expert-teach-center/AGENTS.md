# expert-teach-center Agent Navigation

This folder is a multi-repository area for Expert Teach Center projects.

## Repositories

- `expert-path-app/` - frontend web app (Vite + React + TypeScript).
- `expert-teach-center-vebview/` - mobile/webview shell repository.

## Workflow Rules

- Work in exactly one repository at a time.
- Before implementation, open that repository's local `AGENTS.md` and then relevant `docs/*` files.
- Do not use repository `README.md` as source of truth when `AGENTS.md/docs` exist.
- Avoid cross-repository changes unless explicitly requested.

## Quick Entry Points

If task is frontend UI/business logic in web app:
- Start at `expert-path-app/AGENTS.md`.

If task is app shell/platform/webview integration:
- Start at `expert-teach-center-vebview/AGENTS.md`.
