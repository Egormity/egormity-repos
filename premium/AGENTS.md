# Premium Project Group Navigation

This folder contains Premium CRM client applications.

## Repositories

- `premium-crm-web/` - Vite/React PWA CRM web application with Redux Toolkit, Axios/RTK Query, React Router, and Feature-Sliced style structure.
- `premium_service_v2/` - React Native mobile service app with Redux Toolkit, native Android/iOS projects, Firebase messaging, catalog/order flows, and offline/local storage helpers.

## Agent Rule

For implementation work, enter the relevant repository and read its local `AGENTS.md`. Repository `README.md` files are only short developer quickstarts.
